Detailed Description

The class CharArray wrap the array of primitive type gchar in the object.

Note
This class provide the random access to elements.

Obtain newly created CharArray instance representing the sequence of chars from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of CharArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: CharArray::ofRange(1,5) produces [1, 2, 3, 4].

Obtain newly created CharArray instance representing the sequence of chars from firstValue ( inclusive) to limit ( exclusive) by step offsetByValue.

Note
