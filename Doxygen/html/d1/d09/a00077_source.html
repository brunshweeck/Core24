328 /**
329 * Compares two strings lexicographically.
330 * The comparison is based on the Unicode value of each character in
331 * the strings. The character sequence represented by this
332 * @c String object is compared lexicographically to the
333 * character sequence represented by the argument string. The result is
334 * a negative integer if this @c String object lexicographically precedes
335 * the argument string. The result is a positive integer if this @c String
336 * object lexicographically follows the argument string. The result is zero
337 * if the strings are equal; @c compareTo returns @c 0 exactly when
338 * the @c equals(Object) method would return @c true.
339 *
340 * This is the definition of lexicographic ordering. If two strings are
341 * different, then either they have different characters at some index
342 * that is a valid index for both strings, or their lengths are different,
343 * or both. If they have different characters at one or more index
344 * positions, let @a k be the smallest such index; then the string
345 * whose character at position @a k has the smaller value, as
346 * determined by using the @c < operator, lexicographically precedes the
347 * other string. In this case, @c compareTo returns the
348 * difference of the two character values at position @c k in
349 * the two string -- that is, the value:
350 * @code this.charAt(k)-anotherString.charAt(k) @endcode
351 * If there is no index position at which they differ, then the shorter
352 * string lexicographically precedes the longer string. In this case,
353 * @c compareTo returns the difference of the lengths of the
354 * strings -- that is, the value:
355 * @code this.length()-anotherString.length() @endcode
356 *
357 * For finer-grained String comparison, refer to @c text::Collator.
358 *
359 * @param anotherString the @c String to be compared.
360 * @return the value @c 0 if the argument string is equal to
361 * this string; a value less than @c 0 if this string
362 * is lexicographically less than the string argument; and a
363 * value greater than @c 0 if this string is
364 * lexicographically greater than the string argument.
365 */
366 gint compareTo(String const& anotherString) const override;
