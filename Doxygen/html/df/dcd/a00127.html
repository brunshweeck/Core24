Detailed Description

The class DoubleArray wrap the array of primitive type gdouble in the object.

Note
This class provide the random access to elements.

gdouble limit )
static

Obtain newly created DoubleArray instance representing the sequence of doubles from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of DoubleArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: DoubleArray::ofRange(1,5) produces [1, 2, 3, 4].
