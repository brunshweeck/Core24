Detailed Description

The class ByteArray wrap the array of primitive type gbyte in the object.

Note
This class provide the random access to elements.

static

Obtain newly created ByteArray instance representing the sequence of bytes from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of ByteArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: ByteArray::ofRange(1,5) produces [1, 2, 3, 4].
