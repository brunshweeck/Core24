Detailed Description

The class CharSequence represent the readable sequence of chars values.

this class provide the uniforme read-only access to many different kinds of chars sequences.

A char value represents a character in the Basic Multilingual Plane (BMP) or surrogate.

◆ charAt()

virtual gchar core::CharSequence::charAt ( gint index) const
pure virtual

Returns the char value at the specified index.

An index ranges from zero to length() - 1. The first char value of the sequence is at index zero, the next at index one, and so on, as for array indexing.

If the char value specified by the index is a surrogate, the surrogate value is returned.
