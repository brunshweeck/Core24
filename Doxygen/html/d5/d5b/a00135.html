Detailed Description

The class IntArray wrap the array of primitive type gint in the object.

Note
This class provide the random access to elements.

Obtain newly created IntArray instance representing the sequence of integers from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of IntArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: IntArray::ofRange(1,5) produces [1, 2, 3, 4].

Obtain newly created IntArray instance representing the sequence of integers from firstValue ( inclusive) to limit ( exclusive) by step offsetByValue.

Note
