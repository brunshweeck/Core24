◆ decode()

static Short core::Short::decode ( String const & nm)
static

Decodes a String into a Short.

Accepts decimal, hexadecimal, and octal numbers given by the following grammar:

Decodable String:

  • Sign ( opt) DecimalNumeral
  • Sign ( opt) 0x HexDigits
  • Sign ( opt) 0X HexDigits
  • Sign ( opt) # HexDigits
  • Sign ( opt) 0 OctalDigits
  • Sign ( opt) 0b BinaryDigits
  • Sign ( opt) 0B BinaryDigits

Sign:

  • -
  • +

The sequence of characters following an optional sign and/or radix specifier ("0x", "0X", "0b", "0B" "#", or leading zero) is parsed as by the Short::parseShort method with the indicated radix (10, 16, or 8). This sequence of characters must represent a positive value or a NumberFormatException will be thrown. The result is negated if first character of the specified String is the minus sign. No whitespace characters are permitted in the String.

Parameters
nmthe String to decode.
Returns
a Short object holding the gshort value represented by nm
Exceptions
NumberFormatExceptionif the String does not contain a parsable gshort.

static gshort core::Short::parseShort ( String const & s)
static

Parses the string argument as a signed decimal gshort.

The characters in the string must all be decimal digits, except that the first character may be an ASCII minus sign '-' ('\u002D ') to indicate a negative value or an ASCII plus sign '+' ('\u002B ') to indicate a positive value. The resulting gshort value is returned, exactly as if the argument and the radix 10 were given as arguments to the Short::parseShort(String, int) method.

Parameters

Returns
the gshort value represented by the argument in decimal.

◆ parseShort() [2/2]

static gshort core::Short::parseShort ( String const & s,
gint radix )
static

Parses the string argument as a signed gshort in the radix specified by the second argument.

The characters in the string must all be digits, of the specified radix (as determined by whether Character::digit(gchar,gint) returns a non negative value) except that the first character may be an ASCII minus sign '-'('\u002D') to indicate a negative value or an ASCII plus sign '+' ('\u002B ') to indicate a positive value. The resulting gshort value is returned.

An exception of type NumberFormatException is thrown if any of the following situations occurs:

  • The first argument is null or is a string of length zero.
  • The radix is either smaller than Character::MIN_RADIX or larger than Character::MAX_RADIX.
  • Any character of the string is not a digit of the specified radix, except that the first character may be a minus sign '-' ('\u002D ') or plus sign '+' ('\u002B ') provided that the string is longer than length 1.
  • The value represented by the string is not a value of type gshort.
Parameters
s the String containing the gshort representation to be parsed
radix the radix to be used while parsing s
Returns
the gshort represented by the string argument in the specified radix.
Exceptions
NumberFormatExceptionIf the String does not contains a parsable gshort.

static gint core::Short::toUnsignedInt ( gshort x)
static

Converts the argument to an int by an unsigned conversion.

In an unsigned conversion to an int, the high-order 16 bits of the int are zero and the low-order 16 bits are equal to the bits of the gshort argument.

Consequently, zero and positive gshort values are mapped to a numerically equal int value and negative gshort values are mapped to an int value equal to the input plus 216.

Returns a Short instance representing the specified gshort value.

If a new Short instance is not required, this method should generally be used in preference to the constructor Short::Short(short), as this method is likely to yield significantly better space and time performance by caching frequently requested values.

This method will always cache values in the range -128 to 127, inclusive, and may cache other values outside of this range.

Returns a Short object holding the value given by the specified String.

The argument is interpreted as representing a signed decimal gshort, exactly as if the argument were given to the Short::parseShort(String) method. The result is a Short object that represents the gshort value specified by the string.
