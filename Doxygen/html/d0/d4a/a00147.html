Detailed Description

The class LongArray wrap the array of primitive type glong in the object.

Note
This class provide the random access to elements.

Obtain newly created LongArray instance representing the sequence of longs from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of LongArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: LongArray::ofRange(1,5) produces [1, 2, 3, 4].

Obtain newly created LongArray instance representing the sequence of longs from firstValue ( inclusive) to limit ( exclusive) by step offsetByValue.
