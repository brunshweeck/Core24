Detailed Description

The class ShortArray wrap the array of primitive type gshort in the object.

Note
This class provide the random access to elements.

Obtain newly created ShortArray instance representing the sequence of shorts from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of ShortArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: ShortArray::ofRange(1,5) produces [1, 2, 3, 4].
