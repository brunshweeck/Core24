Detailed Description

The class FloatArray wrap the array of primitive type gfloat in the object.

Note
This class provide the random access to elements.

gfloat limit )
static

Obtain newly created FloatArray instance representing the sequence of floats from firstValue ( inclusive) to limit ( exclusive) by step 1.

Note
The call of FloatArray::ofRange(i,l) produces [i, i+1, i+2, ..., l-1]. For example: FloatArray::ofRange(1,5) produces [1, 2, 3, 4].

Obtain newly created FloatArray instance representing the sequence of floats from firstValue ( inclusive) to limit ( exclusive) by step offsetByValue.

Note
