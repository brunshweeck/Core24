Detailed Description

The Boolean class wraps a value of the primitive type gbool in an object.

An object of type Boolean contains a single field whose type is gbool.

◆ parseBoolean()

static gbool core::Boolean::parseBoolean ( String const & s)
static

Parses the string argument as a boolean.

The boolean returned represents the value true if the string argument is equal, ignoring case, to the string "true". Otherwise, a false value is returned.

Example:

Boolean::parseBoolean("True") // returns true.
Boolean::parseBoolean("yes") // returns false.
